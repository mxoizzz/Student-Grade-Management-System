# 🎓 Student Grade Management System

A menu-driven **CLI (Command Line Interface)** application built using **Core Java** that allows users to manage student records and perform basic grade analysis.

This project was built as part of **PROJECT ASCEND** to strengthen Java fundamentals before learning Object-Oriented Programming (OOP).

---

## ✨ Features

- ➕ Add Student
- 📋 Display All Students
- 🔍 Search Student by Name
- ✏️ Update Student Marks
- 🗑️ Delete Student
- 📊 Calculate Average Marks
- 📈 Find Highest Marks
- 📉 Find Lowest Marks
- 🚪 Exit Application

---

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/mxoizzz/Student_Grade_Management_System.git
```

2. Navigate to the project directory

```bash
cd Student_Grade_Management_System
```

3. Compile the program

```bash
javac Main.java
```

4. Run the application

```bash
java Main
```
